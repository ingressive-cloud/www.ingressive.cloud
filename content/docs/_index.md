# Ingressive Cloud
Welcome to the Ingressive documentation. Ingressive is a cloud load balancer, designed to replace the edge of your network with our global, instantly configurable network. Ingressive handles the whole stack: certificates, DNS, IP Addresses and routing over our global, self healing mesh network. 

## Getting Started
- Add a [Domain](./domain): Create and configure a domain in Ingressive Cloud to start routing traffic.
- Add a Site: Discover how to set up a site for specific subdomains and start accelerating your content quickly.

## Key Features
- Global Edge Network: Benefit from our extensive network of edge servers strategically located around the world.
- Encrypted Mesh: Your traffic flows from our edge network, over an encrypted, self optimising mesh network, directly to the connector in your origin.
- DNS Management: Simplify your DNS configuration with Ingressive Cloud's easy-to-use management tools.
- Performance Optimization: Experience faster load times and improved performance for your web applications.
